# Reality Check — Qué es real y qué falta

## Real / integrado

- Grupos reales del Mundial 2026.
- Formato real: 12 grupos, pasan 2 primeros + 8 mejores terceros.
- Simulación Monte Carlo de partido.
- Simulación de grupos.
- Simulación de mejores terceros.
- Capa de cuotas.
- Comparación entre casas.
- Histórico/CLV.
- Base SQLite inicial.
- Backtesting scaffold.
- Eliminatorias MVP.

## Parcial / baseline

- Ratings por equipo: baseline v1, no modelo final.
- Ranking FIFA: integrado parcialmente con calidad de dato.
- Elo: integrado parcialmente.

## Regla NOVA

Nunca presentar un dato estimado como dato oficial.
Nunca presentar una simulación como certeza.
Toda recomendación debe indicar si falta información.
